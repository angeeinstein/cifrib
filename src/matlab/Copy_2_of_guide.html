Getting Started Guide

Step 1: Add the Bar

Enter the desired length of the bar in the “Length” field under the Bar Setup panel. Click “Add” to place the bar in the workspace. If you want to evaluate safety, you can also select a material here; otherwise, keep the default "Standard" option.

Step 2 (Optional): Define Bar Profile

Select one of four bar profiles under the “Bar Profile” tab. This is used to compute the safety factor against yielding and fracture.

Step 3: Define Supports

Select the type of support (e.g., clamped, roller, or pinned) from the Support panel. Specify the position and click “Add.”

Step 4: Add Joints

Joints allow for specific movement conditions in the structure. You can add rotational joints, shear force joints, or normal force joints. Specify the joint position in the Joints panel and click “Add” to place it.

Step 5: Add Forces and Moments

Specify the force magnitude, angle, and position in the “Loads” panel. Click “Add” to apply the force or moment.

Step 6: Add Distributed Load

Select constant, linear, or quadratic load. Enter the start and end positions, a start value (which can be left at 0), and a pitch. The pitch is calculated as Δy/Δx, where Δx is the distance between start and end positions, and Δy is the maximum variation in load. If a nonzero start value is also specified, Δy is calculated as qmax minus the start value.

Step 7: Verify and Calculate

The calculation can only proceed if the system is in static equilibrium. This is indicated by the status lamp and numerical values in the bottom right corner. If the indicator is green, click “Calculate” to proceed.

Step 8: View the Results

After calculation, you can input a specific position in the bottom right panel to view corresponding values for normal force, shear force, and moment.

Additional Notes

You can undo or redo actions using the Edit menu at the top. This allows you to easily revert or restore changes during setup.

To save your current setup, use the Save option in the File menu. You can later open saved files to reload the setup, avoiding the need to input everything again manually. This also allows you to share project files with others.

There are four example setups available in the File menu, which can help you get started quickly.

The Export option in the File menu lets you generate a PDF containing your setup and results, making it easy to document your work.

For quick access to this guide at any time, navigate to the Help menu.
