Erste Schritte Anleitung

Schritt 1: Den Balken hinzufügen

Geben Sie die gewünschte Länge des Balkens im Feld "Länge" im Balken-Setup-Panel ein. Klicken Sie auf "Hinzufügen", um den Balken in den Arbeitsbereich zu platzieren. Wenn Sie die Sicherheit bewerten möchten, können Sie hier auch ein Material auswählen, andernfalls behalten Sie die Standardoption "Standard" bei.

Schritt 2 (Optional): Balkenprofil definieren

Wählen Sie eines von vier Balkenprofilen im Tab "Balkenprofil" aus. Dies wird zur Berechnung des Sicherheitsfaktors gegen Fließen und Bruch verwendet.

Schritt 3: Lager definieren

Wählen Sie die Art des Lagers (z. B. fest, Rolle oder Gelenk) im Lager-Panel aus. Geben Sie die Position an und klicken Sie auf "Hinzufügen".

Schritt 4: Gelenke hinzufügen

Gelenke ermöglichen bestimmte Bewegungsbedingungen in der Struktur. Sie können Drehgelenke, Querkräfte-Gelenke oder Normalkraft-Gelenke hinzufügen. Geben Sie die Gelenkposition im Gelenke-Panel an und klicken Sie auf "Hinzufügen", um es zu platzieren.

Schritt 5: Kräfte und Momente hinzufügen

Geben Sie die Größe, den Winkel und die Position der Kraft im "Lasten"-Panel an. Klicken Sie auf "Hinzufügen", um die Kraft oder das Moment anzuwenden.

Schritt 6: Verteilte Last hinzufügen

Wählen Sie eine konstante, lineare oder quadratische Last. Geben Sie die Start- und Endpositionen, einen Startwert (der auf 0 bleiben kann) und eine Steigung ein. Die Steigung wird als Δy/Δx berechnet, wobei Δx der Abstand zwischen Start- und Endposition ist und Δy die maximale Laständerung darstellt. Falls ein Startwert ungleich null angegeben ist, wird Δy als qmax minus Startwert berechnet.

Schritt 7: Überprüfen und Berechnen

Die Berechnung kann nur durchgeführt werden, wenn das System sich im statischen Gleichgewicht befindet. Dies wird durch die Statuslampe und die numerischen Werte in der unteren rechten Ecke angezeigt. Wenn die Anzeige grün ist, klicken Sie auf "Berechnen", um fortzufahren.

Zusätzliche Hinweise

Sie können Aktionen über das Menü "Bearbeiten" oben rückgängig machen oder wiederholen. Dadurch können Änderungen im Setup leicht rückgängig gemacht oder wiederhergestellt werden.

Verwenden Sie die Speicheroption im Datei-Menü, um Ihr aktuelles Setup zu speichern. Später können Sie gespeicherte Dateien öffnen, um das Setup wiederherzustellen, wodurch eine erneute manuelle Eingabe vermieden wird. Dies ermöglicht auch das Teilen von Projektdateien.

Im Datei-Menü stehen vier Beispielprojekte zur Verfügung, um Ihnen den Einstieg zu erleichtern.

Die Exportfunktion im Datei-Menü ermöglicht das Erstellen einer PDF-Datei mit Ihrem Setup und den Ergebnissen, um Ihre Arbeit zu dokumentieren.

Über das Hilfe-Menü können Sie jederzeit auf diese Anleitung zugreifen.
